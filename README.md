# ハイパースペクトル

# 実験方法
## 使用したもの
### カメラ
- 一眼レフカメラ（Canon）
  - 有効画素数 1800万画素
  - 撮像素子形式　CMOSセンサー 
- ハイパースペクトルカメラ（SPECIM製）
  - 波長域 400-1000nm
  - 波長分解能 10nm
  - スリット幅 80um

### そのほか
- 温湿度計（Inkbird 製 IBS-TH1 PLUS）
- かぼちゃ（種子：えびす）
- 防除
  - 家庭園芸用カリグリーン
  - 殺菌剤STダコニール 
 
## 撮影
#### 苗
- 北大温室：9-11月で撮影予定（去年も9-11月）
  - 苗に番号を付けて時間軸を追う
  - 週に一度防除をする 
  - RGB：苗とグレーペーパーを一緒に撮影
  - ハイパー：RGBと同じ
#### 圃場
- 北大畑：6月後半から撮影？
- 北農研：北大畑より2週間早く植えている
- 圃場での撮影
  - RGB：静止画と動画の両方
  - ハイパー：ガチ圃場は難しいので苗を並べた仮圃場を作り押し車にハイパーを取り付けて走らせる？
